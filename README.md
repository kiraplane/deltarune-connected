# Deltarune Connected Wiki

Independent English guide site for the Roblox experience Deltarune: Connected at
`https://www.deltarune-connected.wiki`.

The project uses Next.js, OpenNext, and Cloudflare. Its public pages cover:

- Character selection, action menus, hidden morph access, and roster caveats
- Maps, commands, current game status, and mobile or console differences
- The Roaring Knight, Woody, and the Lightners Live update
- Verified Roblox and Viatrix links with explicit wrong-game disambiguation
- An honest code-watch page that does not invent codes or a redeem system

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
```

## Content Rules

- Do not publish unverified redeem codes or claim a redeem system exists.
- Keep DELTARUNE, DONTFORGET CONNECTED, mods, and unrelated Roblox experiences separate from this game.
- Prefer the official Roblox page, Roblox API data, source-attributed community documentation, and recent guide videos.
- Keep the site clearly labeled as an independent guide hub.
- Do not provide scripts, executors, APK mirrors, or account-risk shortcuts.
